  • Cloudstaff Action Scan v1.0.18

    Action Tag Scanner The Action Tag Scanner is designed to scan Action Tags stickers which are available from the http://www.ActionTag.io web site. You can login to the ActionTag.io website and define the event that occurs when an ActionTag is scanned. Supported events include:- Send an SMS, Send an Email or Redirect to a Web Site.
